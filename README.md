# Geoscience Data Analytics & Machine Learning

This repository applies the **Machine Learning for Geoscientists** curriculum by John Lab (Imperial College London) to an independent analysis of Arctic sea ice dynamics. It showcases the transition from manual data handling to automated, reproducible Python workflows designed to extract physical insights from complex polar environments.

## Project Overview

The primary focus of this analysis was to predict summer sea ice flux — the daily change in ice extent. Using a **Lean Modeling** philosophy, the project isolates the most critical physical drivers — solar cycles and momentum — to build a transparent and interpretable predictive system.

The dataset is the publicly available Arctic sea ice extent record sourced from Kaggle.

## Independent Applied Project

**Arctic Sea Ice Flux: A Machine Learning Exploration**

A full machine learning pipeline applied to Arctic sea ice extent data (1979–2020), including:

- Exploratory data analysis and visualization
- Seasonal filtering and chronological train/test splitting
- Scaling methods: standardization (Z-score) and normalization (Min-Max)
- Pipeline construction and model comparison: Linear Regression, Polynomial Regression, and Random Forest
- Feature engineering including Delta-GBM logic for flux prediction
- Model validation and physical plausibility assessment
