# Geoscience Data Analytics & Machine Learning
This repository documents my progress through the ** Data Science and Machine Learning for Geoscientists** curriculum. I used an online available 
data set "arctic sea ice" as a training data set. It showcases the transition from manual data handling to automated, reproducible Python workflows designed to extract physical insights from complex polar environments.

Project Overview
The primary focus of this research was to predict summer sea ice "Delta" (the daily change in ice extent). By utilizing a "Lean Modeling" philosophy, the project isolates the most critical physical drivers—Solar Cycles and Momentum—to build a transparent and interpretable predictive system.
